# Health Habit Tracker 🌿

> A web application to track daily health habits, analyze patterns, and get weekly insights to build a healthier lifestyle.

---

## Phase 1 — Java OOP Structure ✅

### What we built
- `HabitItem` — Abstract class with common fields and abstract methods
- `HabitType` — Enum with 5 habit categories
- `Trackable` — Interface for logging and streak tracking  
- `Analyzable` — Interface for pattern detection and weekly score
- `HabitTracker` — Core class implementing both interfaces
- 5 Habit classes — Water, Sleep, Exercise, Screen Time, Reading
- Console menu — log daily habits and view progress

### Java Concepts Used
Abstract class · Inheritance · Interface · Polymorphism · Encapsulation · Enum · ArrayList · Scanner
